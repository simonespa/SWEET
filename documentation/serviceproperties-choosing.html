Service Properties Choosing Details

This is a flexible operation. It means that the user can chose every kind of word which for him is descriptive, suitable for this web service description.
  1. The user highlights a keyword into the text.
  2. Right clicks on it and choose the "invoke watson" operation.
  3. The string is retrieved and is sent to watson
  4. Watson takes this string in input and returns a list of URL ontologies
  5. This list is showed in a panel, situated on the left side of the editor, but in another tab called "Service Properties Panel".

The URLs retrieved and returned by watson, are shown in a tree view representation. Every direct child of the tree's root element have the name of the string that the user have chosen and next there is a number which said in how many ontologies it is. The children of each item are just the URLs of the ontologies plus the URL of the selected item.


EVENT Details

Service Properties Panel

Is the panel placed on the left side of the editor. It is in a different tab respect to hREST tags Panel and hREST DOM Panel. When the user double cliks on one of the service properties item, it will collapse itself and will show the list of the ontologies URLs plus its URL. If the user clicks on a ontology URL, the description of this ontology will be showed, else if the user clicks on the item's URL, will happen the same think, but in this case, the item will be highlighted.

  • Double Click EVENT

    • Service properties item
      1. It will collapse itself and will show the list of the ontologies URLs.
    • Ontology's URL item
      1. A new view will be showed. This view will be a new Environment where the user can see the whole ontology.
